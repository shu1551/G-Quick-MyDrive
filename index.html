G-クイック 利用者マニュアル

G-クイックは、Google ドライブの操作を「爆速・直感的・高機能」に行うための次世代Webアプリケーションです。ファイルの一覧を更新日順に作成します。基本的な操作として一覧をクリックで選択、ダブルクリックでファイルが開くようになっています。このマニュアルでは、初めて利用する際のセットアップから、日々の業務を効率化する高度な機能までを解説します。

1. 初回セットアップ（重要）

G-クイックを初めて開く際、Googleによるセキュリティ認証プロセスが必要です。

① Googleアカウントでのログイン

アプリのURLを開いた際、Googleアカウントにログインしていない場合はログイン画面が表示されます。操作が必要なアカウントでログインしてください。

② アプリの承認（初回のみ）

ログイン後、Google Apps Script の「承認」画面が表示されます。

  • 「このアプリは Google で確認されていません」 と表示された場合：
    1. 左下の 「詳細」 をクリックします。
    2. 表示された 「G-クイック（安全ではないページ）に移動」 をクリックします。
  • 権限の許可: ドライブのファイル操作などの権限が表示されます。一番下の 「許可」 をクリックします。
NOTE: この承認はGoogleの規定に基づくものです。G-クイックが許可なくデータを外部送信することはありません。

2. メイン画面の構成

● ヘッダーエリア

  • ロゴ (G クイック): アプリのタイトルでクリックで更新されます。
  • 絞り込み: 文字入力で即座にリストをフィルタリングします。
  • スタンプ (★): よく使うフォルダや条件へ、ワンクリックで移動するためのショートカットです。
  • 種類 (三): ドキュメント、スプレッドシート、PDFなど、ファイル形式で絞り込めます。
  • フォルダ: フォルダ階層を潜っていく（ドリルダウン）形式で素早く移動できます。
  • 全リセット (↻): 絞り込み条件（検索ワードやフォルダ選択など）を解除して一覧を広く表示します。
  • 新規作成 (＋): フォルダ作成、ファイルのアップロード、Googleドキュメント等の作成が可能です。

3. 各メニューの機能説明

● 選択メニュー（青色のボタン）

左端をクリックして選択状態にしたファイルに対し、以下の操作が可能です。

  • 選択解除: 選択をすべて解除します。
  • 移動: 選択ファイルを別のフォルダへ移動します。
  • 削除: 選択ファイルをゴミ箱へ移動します。
  • コピー: 選択ファイルを複製します。
  • リンクコピー: 選択ファイルのファイル名にGoogle DriveのURLを埋め込んだリンクをクリップボードにコピーします。複数選択時は改行で縦に並びます。GmailやGoogleドキュメントへのペーストに便利です。

● クイック操作（ファイル名横のアイコン）

各ファイル名の右側に並ぶアイコンから、そのファイルに対して即座に操作を行えます。

  • 🟡 フォルダ表示: そのファイルが入っているフォルダのみにリストを絞り込みます。フォルダ選択時にクリックで絞り込みが解除されます。
  • ✏️ 名前変更: ファイル名やフォルダ名をその場で書き換えます。
  • ⬜ コピー: ファイルを即座に複製します。
  • 🔽 ダウンロード: ファイルをPCにダウンロードします（形式は自動変換）。
  • 🔴 削除: そのファイルだけをゴミ箱へ移動します。

● 基本機能メニュー（三本線アイコン）

  • 並び順切替: 更新日順・作成日順を切り替えます。
  • スタンプ管理: 絞り込み用ショートカットの編集。
  • フォルダ移動: フォルダ階層を移動します。
  • ゴミ箱を表示: ゴミ箱内のファイルを確認・復元します。
  • フォルダを削除: 現在のフォルダをゴミ箱へ移動します。

● 設定メニュー（歯車アイコン）

  1. マニュアルを表示: このヘルプ画面を表示します。
  2. スタートフォルダ切替: 一覧の起点となるマイドライブ内のフォルダを変更します。
  3. 除外フォルダ設定: 一覧に表示しないフォルダを指定します。
  4. ダークモード切替: 外観テーマを変更します。
  5. キャッシュフルリセット: ブラウザキャッシュを完全クリアして最新状態に戻します。

4. キーボード操作（PC）

パソコンではキーボードだけでファイルリストを操作できます。フォーカス中のアイテムは黄色の枠で表示されます。

↑ / ↓ ファイルを一つずつ移動
Enter ファイルを新タブで開く
Space ファイルを選択 / 解除
PageUp / PageDown 1ページ分ずつ移動
Home / End 先頭 / 末尾に移動
/ 検索欄にフォーカス
Escape 検索欄から離脱 / モーダルを閉じる

5. 高度な機能とTips

● Ghost Cache: ブラウザキャッシュと軽量データ構造により、開いた瞬間、爆速でリストを表示します。

● リンクコピーの活用: 複数ファイルを選択してリンクコピーすると、ファイル名にURLが埋め込まれたリンクが縦に並んでコピーされます。GmailやGoogleドキュメントに貼り付けると、クリッカブルなリンク一覧として活用できます。

● 直感操作: 外側クリックや Esc キーでメニューを閉じられます。設定メニュー等はファイル誤選択防止ガードが働きます。
