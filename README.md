# G-クイック（G-Quick MyDrive）

自分の **マイドライブ** のファイルを、フォルダの階層を無視して一枚にフラットに並べ、検索でその場で絞り込む Google Apps Script (GAS) Webアプリ。20年もののExcel製「ファイル一覧」の発想を、そのままGoogleドライブに持ち込んだものです。

## 特徴

- **フラット表示**: マイドライブの全ファイルを、フォルダ階層を無視して一枚の表に並べる
- **瞬時の検索**: データを最初にまとめてブラウザ側へ持ってくるので、文字を打つそばからその場で絞り込み（検索ごとにサーバーと往復しない）
- **自分の領域だけ**: 使う人自身の権限で動き、見えるのは**その人自身のマイドライブだけ**。他人のデータが混ざらない
- **接続は Drive ひとつだけ**: 余計なサービスには一切つながない
- **スタートフォルダ**: よく見るフォルダを起点に設定し、そのURLをブックマークすればワンクリックで開く
- **多彩な絞り込み**: 検索／スタンプ（よく使う条件のショートカット）／種類（ドキュメント・PDF など）／フォルダのドリルダウン／更新日順・作成日順
- **ファイル操作**: 名前変更・コピー・移動・複数選択での一括操作・ごみ箱の復元/完全削除・新規フォルダ/Googleドキュメント作成・アップロード
- **キーボード操作**: ↑↓ / j k・Enter・Space・Home/End・`/` で完結
- **ダーク/ライトモード**: 切り替え可能

## 設計の考え方（なぜマイドライブだけなのか）

もともとは Drive・Gmail・カレンダー・チャットなど複数の Google サービスにつないだ「何でもできる」版を作っていたのですが、機能を盛りすぎて複雑になり、共有ドライブ対応など安定しないところも出てきました。そこで **「自分が管理したいのは、結局マイドライブだけ」** と割り切り、対象をマイドライブひとつに閉じて作り直したのが、この版です。

接続するサービスは Drive ひとつだけ。使う人自身の権限で動くので、見えるのは **その人自身のマイドライブだけ**で、他人のデータが混ざることはありません。

範囲を「自分の領域」に閉じたことで、設計はシンプルになりました。Googleは全世界の何十億ファイルを相手にする設計を取らざるを得ませんが、「自分のマイドライブだけ」と割り切ったG-クイックは、**全部を手元に持ってから絞り込む**という、純正には選べない設計を選べます。

## 動作要件

- Google アカウント
- Google Apps Script（無料）
- Drive API v3（GAS のアドバンスドサービスから有効化。`appsscript.json` 同梱済み）

## セットアップ手順

### いちばん簡単：Drive からコピー

[こちらの共有リンク](https://script.google.com/d/1s8SbYIsXobq5oicQfCRYgeGBXcKW3E4hhSo_ms-I8AmwlMffZXJwzJRQ/edit?usp=sharing) を開き、ドライブで**右クリック →「コピーを作成」**するだけ。自分の Google ドライブにプロジェクトごと複製されます（ファイルのコピペ不要）。あとは下の「**3. デプロイ**」以降を行えば使えます。

### 手動でセットアップ（clasp / コピペ派向け）

1. **GAS プロジェクトを作成**
   - [Google Apps Script](https://script.google.com/) で「新しいプロジェクト」
2. **このリポジトリのファイルをコピー**
   - `appsscript.json` / `Code.js` / `index.html` の中身を、それぞれ同名のファイルとして GAS エディタに貼り付ける
   - `appsscript.json` を編集するには、左側の「プロジェクトの設定」→「`appsscript.json` マニフェスト ファイルをエディタで表示する」をオンに
   - 同梱の `appsscript.json` には Drive アドバンスドサービスの有効化が含まれています
3. **デプロイ**
   - 右上「デプロイ」→「新しいデプロイ」
   - 種類: 「ウェブアプリ」
   - 実行: 「自分」
   - アクセスできるユーザー: 「自分のみ」（推奨）
4. **承認**
   - 初回は「承認が必要」と表示される。「アクセスを承認」→ Google アカウントを選択 →「詳細」→「（安全ではないページ）に移動」→「許可」
5. **ブックマーク**
   - 表示された Web アプリの URL をブックマーク。よく使うフォルダを**スタートフォルダ**に設定しておくと、ワンクリックでその一覧が開く

## clasp で開発する場合

```bash
# .clasp.json を作る
cp .clasp.json.example .clasp.json

# .clasp.json の scriptId に、自分の GAS プロジェクトの scriptId を記入
# scriptId は GAS エディタの「プロジェクトの設定」で確認できる
```

## キーボード操作

| キー | 動作 |
|---|---|
| ↑ / ↓（または j / k） | ファイル移動 |
| Enter | 開く（新しいタブ） |
| Space | 選択 / 解除 |
| Page Up / Page Down | ページ送り |
| Home / End | 先頭 / 末尾 |
| / | 検索欄にフォーカス |
| Escape | 検索欄離脱 / モーダルを閉じる |

## 仕様メモ（実装の特徴）

- **Sheets を介在させない**: データをスプレッドシートに置かず、最初にまとめてブラウザ側へ持つ設計。クラウドの往復をなくして検索を高速化
- **フォルダを介在させない**: 全ファイルをフラットに並べ、検索・絞り込みで当てる
- **接続サービスは Drive ひとつ**だけ（`appsscript.json` の `enabledAdvancedServices` も Drive のみ）
